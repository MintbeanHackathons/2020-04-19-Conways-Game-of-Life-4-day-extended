# Conways Game of Life

This is a simple read only version of John Conway's game of life built using React.

Online at [https://g-o-l-2020.herokuapp.com/](https://g-o-l-2020.herokuapp.com/)

run: yarn dev -
to run locally

## Dependencies

- react 16.13.1
- react-dom 16.13.1
- react-scripts 3.4.1
- serve 11.3.0
- styled-components 5.1.0
